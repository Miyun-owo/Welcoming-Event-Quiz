# 檔案操作須知

## 資訊倫理警告
- 本體驗會模擬電腦異常，但不會修改或刪除您的檔案。所有的體驗皆以SFML實踐(純C++)，玩家操作過程中不涉及終端機、批次檔等功能，任何的引用也不可將之改寫為對電腦記憶體的直接操控與指派，否則可能被系統辨識為駭客攻擊(如使用公用電腦，則可能被通報為資安事件)。
- 本檔案所有程式碼僅作為114學年度復旦程式設計班迎新活動使用，若未來有使用本檔案的需求，請聯絡檔案持有者。

## 使用與安裝方式(安裝人員，非玩家)
1. 到SFML官網，對要"執行該功能的電腦"下載SFML。[操作說明](https://www.sfml-dev.org/tutorials/3.0/getting-started/build-from-source/?utm_source#building-sfml)
2. 完成下載後，照[這個說明](https://www.sfml-dev.org/tutorials/3.0/graphics/text/?utm_source)建置ttf.檔案。 這個檔案可以直接被包在Github專案裡面，不用每個電腦都做一次。極度建議不要打中文或一些奇怪符號，用ASCII能表示的字元就好。這個是決定你檔案的文字風格用的。
3. (跟ttf.檔要有共同的父資料夾) 建置一個可執行的cpp.檔案。(檔名為puzzle.cpp)，記得要下載完SFML再來，不然code會報錯。 
(目前這個檔案是GPT給我的，還沒測試過，先不要真的執行，我不知道它會不會跳出什麼嚇到人的東西)
```
我給GPT的prompt:
我想做個謎題
第一個是輸入一個三位數密碼，然後輸入對了之後進入第二階段，如果輸入錯會鎖死，一分鐘之後檔案自己把自己關掉再重開
第二個是惡搞的bash，我想模擬出電腦被駭的感覺，基本上就是視窗亂跳之類的or其他被駭的電腦會出現的狀況，讓人以為電腦壞掉，但其實只是模擬。我希望玩家可以找出一個視窗，該視窗會問玩家一些問題(類似終端機安裝東西時要寫的那種，很簡單但玩家要冷靜找出來就是難度所在，我不希望它太難找，因為受眾多數沒有學過計算機概論之類的知識)
第三個就是解謎部分，回答完問題之後，會跳出像是終端機的頁面，上面有一些真正的謎題(推理之類的東西)，然後玩家要回去原本的電腦去找一些檔案來破解這個問題，類似CTF找flag但再簡單一些。最後結果是會輸出一個link
```
4. 完成之後，把檔案另存下來，存到USB裡面。
5. 活動進行時，務必確保有一台電腦是開著原始碼的，如果有出任何問題可以及時修改。
